## Freeze-B-Gone

Front end UI for the web application developed using React. 

### Prerequisites


```
Node package manager / Yarn 

Webpack 

```

### Installing

Install Yarn then: 

```
Commands: 

yarn install
npm run dev 

```

## Deployment

Open http://localhost:8080/ in browser.
